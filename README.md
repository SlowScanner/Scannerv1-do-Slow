# 🔎 Slow Security Scanner

Scanner de segurança para Android rodando no **Termux**.

Ferramenta criada para analisar o armazenamento do dispositivo e detectar arquivos suspeitos, scripts, APKs e arquivos ocultos.

---

## 📸 Interface

![Scanner Interface](https://i.imgur.com/8Q2QF5H.png)

---

## ⚡ Recursos

- 🔍 Scan completo do storage
- 📦 Detecção de APKs
- 📜 Detecção de scripts
- 👁 Arquivos ocultos
- 🚨 Arquivos suspeitos (mod, hack, cheat)
- 📱 Lista de apps instalados
- 📄 Relatório automático

---

## 📥 Instalação

Clone o repositório:

```bash
git clone https://github.com/SlowScanner/Scannerv1-do-Slow
```

Entre na pasta:

```bash
cd Scannerv1-do-Slow
```

Execute o instalador:

```bash
bash install.sh
```

---

## ▶️ Executar scanner

```bash
./scanner.sh
```
